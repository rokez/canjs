can-set.Algebra.prototype.getUnion

algebra.getUnion(a, b, aItems, bItems)

Unifies items from set A and setB into a single array of items.

algebra = new set.Algebra(
  set.comparators.rangeInclusive("start","end")
);
algebra.getUnion(
  {start: 1,end: 2},
  {start: 2,end: 4},
  [{id: 1},{id: 2}],
  [{id: 2},{id: 3},{id: 4}]);
  //-> [{id: 1},{id: 2},{id: 3},{id: 4}]

Parameters

  1. a {can-set.set}:

    A set.

  2. b {can-set.set}:

    A set.

  3. aItems {Array<Object>}:

    Set a's items.

  4. bItems {Array<Object>}:

    Set b's items.

  • returns {Array<Object>}:

    Returns items in both set a and set b.
