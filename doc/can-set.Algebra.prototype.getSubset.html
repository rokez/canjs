can-set.Algebra.prototype.getSubset

algebra.getSubset(a, b, bData)

Gets a set's items given a super set b and its items.

algebra.getSubset(
  {type: "dog"},
  {},
  [{id: 1, type:"cat"},
   {id: 2, type: "dog"},
   {id: 3, type: "dog"},
   {id: 4, type: "zebra"}]
) //-> [{id: 2, type: "dog"},{id: 3, type: "dog"}]

Parameters

  1. a {can-set.set}:

    The set whose data will be returned.

  2. b {can-set.set}:

    A superset of set a.

  3. bData {Array<Object>}:

    The data in set b.

  • returns {Array<Object>}:

    The data in set a.
