can-set.Algebra.prototype.union

algebra.union(a, b)

Returns a set that represents the union of A and B (A ∪ B).

algebra.union(
  {start: 0, end: 99},
  {start: 100, end: 199},
) //-> {start: 0, end: 199}

Parameters

  1. a {can-set.set}:

    A set.

  2. b {can-set.set}:

    A set.

  • returns {can-set.set|undefined}:

    If an object is returned, it is the union of A and B (A ∪ B).

    If undefined is returned, it means a union can't be created.
