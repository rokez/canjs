{{&key}}

Insert the unescaped value of the key into the output of the template.

{{&key}}

The {{&key}} tag is an alias for {{{key}}}, behaving just like {{key}} and {{helper}} but does not escape the result.

Parameters

  1. key {key}:

    A key that references a value within the current or parent context. If the value is a function or compute, the function's return value is used.

  • returns {String|function|*}:
