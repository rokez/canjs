keys

Returns an array of the map's keys.

Map.keys(map)

var people = new Map({
        a: 'Alice',
        b: 'Bob',
        e: 'Eve'
});

Map.keys(people); // ['a', 'b', 'e']

Parameters

  1. map {can-map}:

    the Map to get the keys from

  • returns {Array}:

    array An array containing the keys from map.
