can-set.Algebra

Creates an object that can perform binary operations on sets with an awareness of how certain properties represent the set.

new set.Algebra(compares...)

Creates an object that can perform binary operations on sets with an awareness of how certain properties represent the set.

var set = require("can-set");
var algebra = new set.Algebra(
  set.comparators.boolean("completed"),
  set.comparators.id("_id")
);

Parameters

  1. compares {compares}:

    Each argument is a compares. These are returned by the functions on can-set.comparators or can be created manually.

  • returns {can-set.Algebra(compares)}:

    Returns an instance of an algebra.
